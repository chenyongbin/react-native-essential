# isIPhoneX  

判断当前机型是否是iPhoneX、iPhone XR、 iPhone XS、iPhone 11等类似机型的方法。  


## @1.0  

### 引入路径  

```js
import { default as isIPhoneX } from "react-native-essential/utils/isIPhoneX/@1.0";
```  

**入参**  

_无_  

**出参**  

| 名称 | 类型 | 说明 | 
| - | - | - | 
| result | `boolean` | iPhoneX、iPhone XR、 iPhone XS、iPhone 11等类似机型 |
