# buildUrl  

## @1.0  

该模块输出一个方法，该方法可以将一个基本url和一个参数对象拼接成最终url的方法。  

### 引入路径  

```js  
import { default as buildUrl } from "react-native-essential/utils/buildUrl/@1.0";
```  

入参  

| 名称 | 类型 | 说明 | 是否必填 | 默认值 |
| - | - | - | :-: | :- | 
| url | `string` | 基本url | 是 | 无 |   
| params | `Object` | 参数对象 | 否 | 无 |   


出参    

一个`string`类型的最终url
